# MainHashSetPalabra 

**Escribe un programa que lea una lista de palabras y muestre solo las palabras
únicas (ignorando las repetidas) usando un HashSet. (MainHashSetPalabra)**

Para hacerlo, he creado un Set de String (obviamente para almacenar las palabras) y para mi Set sirve para ignorar las palabras repetidas.
Despues de esto, añado algunas palabras y las muestro por la pantalla.
